# 🎓 Luminé - Iskolai Adminisztrációs Rendszer

Modern, webalapú iskolai adminisztrációs platform diákok, tanárok és adminisztrátorok számára.

## ✨ Főbb Funkciók

### 👨🎓 Diák
- 📅 Órarend megtekintése
- 📊 Jegyek és átlagok nyomon követése
- ✅ Mulasztások áttekintése
- 📝 Házi feladatok kezelése
- 🎵 Zene kérés az iskolai rádióba
- 💬 Üzenőfal
- 📱 QR kód be/kilépéshez

### 👨🏫 Tanár
- ✍️ Jegyek beírása
- 📋 Mulasztások rögzítése
- 📚 Házi feladatok kiadása
- 🔄 Helyettesítések kezelése

### 👨💼 Admin
- 👥 Felhasználók kezelése
- 🗓️ Órarend szerkesztése
- 🏫 Osztályok kezelése
- 📈 Rendszerstatisztikák

## 🚀 Technológiák

- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Tesztelés**: Vitest, Playwright
- **Deployment**: Vercel
- **Ikonok**: Lucide-react

## 📦 Telepítés

```bash
npm install
cp .env.example .env.local
npm run dev
```

## 🧪 Tesztelés

```bash
npm run test
npm run test:e2e
npm run test:all
```
